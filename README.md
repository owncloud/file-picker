# ownCloud File picker
Easily integrate ownCloud into your own web applications.

## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm serve
```

### Compiles and minifies for production
```
pnpm build
```

### Run your unit tests
```
pnpm test:unit
```

### Lints and fixes files
```
pnpm lint
```
